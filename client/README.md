# Portico Client

React frontend for Portico bookmark manager, built with Vite, TypeScript, and Tailwind CSS v4.

## Setup

```bash
bun install
```

## Development

```bash
bun run dev
```

Server runs on `http://localhost:5173`

## Build

```bash
bun run build
```

Outputs to `dist/` directory for embedding in the Go server.

## Preview

```bash
bun run preview
```

## Technologies

- **React 19** - UI framework
- **Vite 7** - Build tool
- **TypeScript** - Type safety
- **Tailwind CSS v4** - Styling
- **Bun** - Package manager and runtime
